## ✨ Features

- **52 curated sites** across 5 categories: Career & Education, Content Design, Productivity, Utilities, Fun
- **Live search** — search by name, tags, use case, or URL (keyboard shortcut: `/`)
- **Category filters** — one-click to drill into any category
- **Sort** — alphabetical, reverse, or by category
- **Animated grid** with staggered fade-in on load and scroll
- **Favicons** auto-loaded from Google's favicon service
- **Fully accessible** — ARIA labels, keyboard navigation, focus indicators
- **Mobile-responsive** — single-column layout on small screens
- **Reduced-motion** support

---

## 🖥️ Deploying to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site will be live at `https://<username>.github.io/<repo>/`

No build step required — pure HTML, CSS, JavaScript.
